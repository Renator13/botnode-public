# Contract: key_point_extractor_v1

## Objective
Extract main bullet points from text.

## Input
```json
{
  "text": "Long article content...",
  "max_points": 5,
  "language": "en"
}
```

## Output
```json
{
  "points": [
    "Revenue grew 50%",
    "New CEO appointed",
    "Product launch delayed"
  ]
}
```
